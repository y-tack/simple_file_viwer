シンプルファイル閲覧機 ? Ver 0.5

このアプリは、テキストファイル（各種文字コード）や16進数バイナリファイルを簡単かつ快適に読み込むための、Python製デスクトップ閲覧ツールです。
Ver 0.5では、使いやすさを追求した「数字入力ダイアログ」と、長文の読み込みに便利な「前へ・次への双方向検索」を新しく追加しました。

1. 主な機能
文字コード自動判定: Shift-JIS、UTF-8、UTF-16を自動で判別して綺麗に表示します。

16進数バイナリ表示: テキストとして読めないファイルも、16進数のバイナリデータに変換して中身を確認できます。

双方向検索（★新機能）: 「次へ（下方向）」だけでなく、「前へ（上方向）」も探せる連続検索ボタンを搭載。端まで行くとループ案内が出ます。

ダイアログによる動的設定（★新機能）: 1行の折り返し桁数や、本文の文字サイズを専用の数字入力画面からその場でサクッと変更できます。

自動設定保存: 最後に使っていたフォルダーの位置、折り返し桁数、文字サイズは、アプリを閉じると自動でiniファイルへ保存されます。

2. フォルダ内のファイル構成
プログラムは役割ごとに綺麗に4つのファイルに分かれています。すべて同じフォルダーの中に置いて実行してください。

main.py: アプリの起動口です。画面全体のレイアウトやボタンの配置を行っています。

config.py: 設定を管理します。iniファイルの保存や、見出し（Markdown）の装飾ルールを決めています。

file_processor.py: ファイル処理の専門家です。文字コードの判定やバイナリ変換を担当します。

editor_functions.py: アプリの頭脳です。検索の仕組みや、ダイアログが開いたときの動きをコントロールします。

3. アプリの使い方・操作手順
① アプリの起動
コマンドプロンプトやターミナルを開きます。

4つのファイルが置かれているフォルダーへ移動し、以下のコマンドを実行します。

Pythonをインストールして
cdコマンドでファイルを置いてあるフォルダーへ移動し
Python main.py
として実行して下さい

② フォルダーの選択とファイルの閲覧
画面最上部の [フォルダーを選択する] ボタンを押し、読みたいファイルが入っているフォルダーを選びます。

左側のリストボックスにファイル一覧が自動で表示されるので、読みたいファイルをダブルクリック、または選択します。

モードを「自動判別」にしておけば、文字コードを気にせず中身が右側のテキストエリアに表示されます。別のコードで強制的に読み直したい場合は、上部のラジオボタンを切り替えてください。

③ 本文の見やすさ調整（★Ver 0.5 新機能）
折り返し桁数の変更:
[桁数を変更] ボタンを押すと入力画面が出ます。「1?200」の間で好きな数字を入れて「OK」を押すと、その文字数で本文が綺麗に折り返されます。

文字サイズの変更:
[文字サイズを変更] ボタンを押し、「10?40」の間で文字の大きさをpt単位で指定します。「OK」を押すと、その瞬間に本文全体の文字の大きさが切り替わります。

④ 双方向検索のしかた（★Ver 0.5 新機能）
画面右上の 検索: の横にある入力欄に、探したいキーワードを入力します。

[次へ] ボタンを押すと、現在のカーソル位置から下に向かって単語を探し、見つかった文字を黄色くハイライトします。

[前へ] ボタンを押すと、現在のカーソル位置から上に向かって（逆方向に）単語を探します。

文章の端（一番下、または一番上）まで探し終えると、「戻って再検索しますか？」と案内が出るので、何度もループして探し続けることができます。続き

4. 安全設計（数値の制限ルール）

入力ダイアログでは、アプリが正常に動くように数値の範囲が制限されています。制限を超えた数字や、文字を入力した場合は自動で弾かれる安全設計です。

折り返し桁数: 1 ? 200 文字の間で入力してください。

文字サイズ: 10 ? 40 pt の間で入力してください。

エラー防止: ダイアログに日本語やアルファベットが入力された場合は、Pythonの機能が自動でエラーを検知し、入力をブロックします。
